# SM Ecosystem

Production-ready React SPA with admin portal, client portal, animations, PWA support.

## 🚀 Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## 🏗️ Build for Production (Vercel)

```bash
npm run build
```

Outputs to `/dist` - drag/drop to Vercel or `vercel deploy`.

## Features

- Cinematic animations & 3 ecosystem themes (Hardscape, Construction, Maintenance)
- Full admin panel (billing, projects, clients, analytics, themes)
- Client portal with documents & schedule
- PWA with service worker & offline page
- React Router SPA routing
- TailwindCSS + Radix UI components

Ready for GitHub → Vercel pipeline.
